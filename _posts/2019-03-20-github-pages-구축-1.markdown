#### 1. Atom, Ruby 설치, jekyll 설치
Atom을 설치하여 로컬에서 블로그를 관리하기로 하였다.
또한 Ruby와 jekyll을 설치하여 로컬에서 github pages를 확인하기 위한 환경을 조성했다.

처음에는 github 웹페이지 내에서 직접 포스트를 push하면 될 것이라 생각했지만
생각보다 많이 불편했다.

- Atom 설치, github 연동
